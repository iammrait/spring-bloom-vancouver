# Screenshots for Spring Bloom in Vancouver

Add PNG captures here and embed them from the root [README.md](../../README.md) **Screenshots** section when you are ready.

Suggested filenames (consistent aspect ratio helps, for example 1200×750):

| File | What to show |
|------|----------------|
| `typescript.png` | TypeScript or TSX: imports, types, strings, keywords |
| `markdown.png` | Markdown: headings, code fence, links |
| `html.png` | HTML or JSX: tags, attributes, punctuation |

**Tips**

- Use the theme **Spring Bloom in Vancouver** and a font you like; avoid minimap clutter if the shot is small.
- macOS: `Cmd+Shift+4` then Space for a window capture. Windows: Snipping Tool or Win+Shift+S.
- Optional: hide sensitive paths or replace folder names before sharing.

This folder is excluded from the packaged `.vsix` (see `.vscodeignore`) so installer size stays small; images remain in Git for the GitHub readme.
