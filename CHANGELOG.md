# Changelog

## [1.0.2] – Branding

- Package `author` URL and README **About** / **Links** now include [X profile @iammrait](https://x.com/iammrait).

## [1.0.1] – Screenshots

- README screenshots (TypeScript, Markdown, HTML) as palette-accurate demo assets in `docs/screenshots/`; regenerate with `docs/screenshots/generate_screenshots.py` (requires Pillow).

## [1.0.0] – Initial release

- Full workbench UI coverage (editor, tabs, activity bar, side bar, panels, terminal, status bar, scrollbars, git decorations, peek view, minimap).
- TextMate `tokenColors` and `semanticTokenColors` for consistent LSP highlighting.
- Palette derived from Vancouver cherry blossom street photography: creamy daylight, blossom pink, sky blue, and grass green.
- Published on [Open VSX](https://open-vsx.org/extension/iammrait/spring-bloom-vancouver) as `iammrait.spring-bloom-vancouver`.
